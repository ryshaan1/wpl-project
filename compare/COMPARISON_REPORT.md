# Experiment 9 – Comparison Report
## Manual Coding vs AI-Generated Code | VoltGrid

---

## Task
Write a PHP function to calculate the total EV charging cost (session fee + 18% GST + ₹5 platform fee).

---

## Comparison Table

| Parameter | Manual | AI-Generated |
|-----------|--------|--------------|
| **Readability** | Simple, easy to read | Clear with comments |
| **Maintainability** | No type hints, harder to maintain | Type hints make it safer |
| **Error Handling** | No input validation | Handles invalid input (returns 0) |
| **Code Standards** | Basic PHP style | Follows PSR-12 standards |
| **Time to Write** | Faster | Slightly slower |

---

## Key Difference
The manual version is shorter but does not handle edge cases (e.g. negative duration).  
The AI version adds input validation and type hints, making it safer for production use.

---

## Conclusion
- **Manual coding** is quicker for simple tasks and easier to understand at a glance.
- **AI-generated code** is more robust and follows better coding practices.
- Best approach: use AI as a starting point, then review and simplify as needed.
